# YouTube Content Analysis Dashboard

## Overview

The **YouTube Content Analysis Dashboard** is an interactive Power BI dashboard designed to analyze YouTube video performance, audience engagement, publishing trends, and content distribution across multiple channels and countries. It provides comprehensive insights into video popularity, upload behavior, and engagement metrics, helping content creators and analysts make data-driven decisions.

The dashboard transforms raw YouTube data into meaningful visualizations, enabling users to identify high-performing content, optimal publishing strategies, and audience engagement patterns.

## Features

* Interactive filters for:

  * Publishing Country
  * Video Title
  * Channel Name
* Overall performance KPIs

  * Total Videos
  * Average Views
  * Total Channels
  * Engagement Rate
* Video performance analysis
* Channel performance comparison
* Upload day performance analysis
* Publishing time impact analysis
* Country-wise video distribution
* Category-wise view distribution
* Comments disabled analysis

---

## Dashboard KPIs

| KPI             | Description                            |
| --------------- | -------------------------------------- |
| Total Videos    | Total number of videos analyzed        |
| Average Views   | Average views received per video       |
| Total Channels  | Number of unique YouTube channels      |
| Engagement Rate | Overall audience engagement percentage |

## Business Insights

* Video performance varies significantly across different content categories and channels.
* Publishing day has a noticeable impact on both total uploads and overall views.
* Certain publishing time windows consistently generate higher view counts.
* A small number of channels contribute a large share of total views, highlighting the presence of high-performing creators.
* Category-wise analysis helps identify content types that attract the highest audience engagement.
* Country-level insights reveal geographical differences in content production and viewership.
